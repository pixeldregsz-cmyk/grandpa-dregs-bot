🤖 What You're Building

A fully autonomous AI-powered Telegram bot that:

  • Runs 24/7 on Fly.io's free tier ($0/month)
  • Has a customizable personality (edit one file)
  • Remembers conversations across reboots (persistent SQLite memory)
  • Can run shell commands, Python code, and make HTTP requests
  • Can search the web, download files, and send files to users
  • Auto-restarts if it crashes
  • Uses free AI from OpenGateway (no API key needed)

This is the same process that was used to build Grandpa Dregs — a live AI bot running on Telegram right now. Built entirely from an iPhone with no computer, no terminal, and no copy-paste.

1 Create a Telegram Bot

  1. Open Telegram on your phone
  2. Search for @BotFather and open the chat
  3. Send /newbot
  4. Choose a display name for your bot (e.g., "My AI Buddy")
  5. Choose a username that ends in "bot" (e.g., myaibuddy_bot)
  6. BotFather will give you a bot token — it looks like:
    123456789:ABCdefGHIjklMNOpqrsTUVwxyz
  7. Save this token — you'll need it later. Don't share it with anyone.
💡 Tip: You can message your bot right now to make sure it exists. It won't respond yet — that's expected.

2 Create a Fly.io Account

  1. Open Safari on your phone
  2. Go to https://fly.io
  3. Tap "Sign Up" or "Get Started"
  4. Create an account with your email
  5. You'll need to add a credit card for verification — this is just to prevent abuse. You will NOT be charged. The free tier covers everything we need.
  6. Once signed in, you should see your Fly.io dashboard
💡 Tip: Fly.io's free tier gives you 3 shared VMs and 3GB of storage. Our bot uses 1 VM and 1GB. You're nowhere near the limit.

3 Create a GitHub Account & Repo

  1. Go to https://github.com in Safari
  2. Sign up for a free account if you don't have one
  3. Once logged in, tap the + icon (top right) → "New repository"
  4. Name it something like my-ai-bot
  5. Keep it Public (easier for Fly.io to access)
  6. Check "Add a README file"
  7. Tap "Create repository"

You now have a place to store your bot code. Next, we'll add the files.

4 Download & Upload the Bot Files

Download each file below, then upload them to your GitHub repo.

How to upload files to GitHub (on iPhone):

  1. Open your repo in Safari (e.g., github.com/YOUR-NAME/my-ai-bot)
  2. Tap "Add file" → "Upload files"
  3. Tap "Choose files" and select the file from your Downloads
  4. Tap "Commit changes"
  5. Repeat for each file

Files to download:

bot.py
The main bot code — the brain with 9 tools
soul.md
Your bot's personality — edit this to change who it is
requirements.txt
Python dependencies
Dockerfile
Container build config
fly.toml
Fly.io deployment config
💡 Tip: Download all 5 files before uploading. Then upload them one at a time to GitHub. Make sure the filenames are exact — lowercase, with the right extensions.

5 Deploy to Fly.io

  1. Go to https://fly.io and log in
  2. Tap "Launch" or "Create App" from the dashboard
  3. Select "GitHub" as the source
  4. Connect your GitHub account if prompted
  5. Select the repo you created (e.g., my-ai-bot)
  6. Fly.io will detect the Dockerfile and fly.toml automatically
  7. Choose a name for your app (e.g., my-ai-bot)
  8. Select a region close to you (e.g., iad for US East)
  9. Before deploying, you need to set secrets (next step)
⚠️ Important: Don't deploy yet! You need to set your Telegram token first (next step).

6 Set Your Secrets

Go to your app's dashboard on Fly.io and set these environment variables (secrets):

Secrets to set
TELEGRAM_TOKEN = [your bot token from Step 1]

That's it! The AI API is already configured (free, no key needed).

How to set secrets on Fly.io (iPhone):

  1. Go to your app's dashboard on Fly.io
  2. Find "Secrets" in the sidebar or menu
  3. Tap "Add secret"
  4. Set TELEGRAM_TOKEN to your bot token from Step 1
  5. Save

Create a storage volume:

  1. In your app's dashboard, find "Volumes"
  2. Tap "Create volume"
  3. Name: bot_memory
  4. Size: 1 GB
  5. Region: same as your app
  6. Create it

Now tap "Deploy" to launch your bot!

7 Test Your Bot

  1. Open Telegram
  2. Search for your bot by its username (e.g., @myaibuddy_bot)
  3. Send /start
  4. It should respond with a welcome message

Try these:

  • Who are you? — tests personality
  • Run whoami on the server — tests shell access
  • What's the latest news about AI? — tests web search
  • Calculate the first 20 prime numbers — tests Python
  • /remember My name is [your name] — tests memory
💡 Tip: If the bot doesn't respond, check the logs in your Fly.io dashboard. Look for "Logs" in the sidebar. Common issues: wrong Telegram token, or the volume isn't mounted.

🎉 You're Live!

Your AI bot is now running 24/7 for free on Fly.io. It will keep running whether you close your phone, turn off Safari, or go to sleep.

✨ Customize Your Bot

Change the personality

Edit soul.md in your GitHub repo. This is your bot's entire personality. You can make it:

  • A pirate captain who codes
  • A zen master chef
  • A sarcastic librarian
  • A Starfleet veteran
  • Whatever you want

After editing, go to Fly.io and deploy again to pick up the changes.

Change the AI model

In your fly.toml, you can change the model:

fly.toml
[env]
  MODEL = "mimo-v2.5-pro"

Add new tools

Edit bot.py to add new capabilities. The bot uses OpenAI-compatible function calling, so you can add any tool you can write in Python.

🔧 Managing Your Bot

View logs

Go to your app on Fly.io → "Logs" in the sidebar. You'll see every message, tool call, and error.

Restart

Go to your app → "Machines" → tap your machine → "Restart"

Update the bot

  1. Edit files on GitHub
  2. Go to Fly.io → your app → "Deploy"
  3. It'll rebuild and restart automatically

Delete everything

Go to your app on Fly.io → "Settings" → "Delete app"
